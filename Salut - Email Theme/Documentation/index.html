Files Structure

The template is organized in the following way:

  • Documentation - Contains documentation and guides of the template.
  • Demo - ... - Main files of the template. This is what you're going to upload on your email serivces.

    • CampaignMonitor - CampaignMonitor compatible file.
    • PSD's - Contains 4 PSD file.
    • Stampready - Stampready compatible file are inside this folder as .html file use this to build an email design on stampready.
    • Custom Stampready - Custom stampready compatible file are inside this folder as .html file use this to build an email design on stampready.
        HTML.zip - Pure HTML file of the template.
        Mailchimp.zip - Mailchimp compatible file.
        Mailster.zip - Mailster compatible file.

Getting Started With Stampready Online Editor

Here are few steps to get started with Stampready:

Before start editing the template on stampready you have to turned off "use new editor" feature on stampready, which I will explain below :

we assuming that you have logged in to your account, click on ACCOUNT > SETTINGS > UNCHECK THE "USE NEW EDITOR"



Next on the home page click on the Campaigns > New Campaign > Import Template > HTML File after that you may have to attach the HTML file of pro email template > Do some necessary adjusement > Export

Getting Started With Campaign Monitor

Here are few steps to get started with CampaignMonitor:

We assuming you have changed image background manually before uploading it to CampaignMonitor (see at the costumization section).

  1. Click your profile image at the top right, then select My templates.
  2. Choose Upload your own HTML (if you have existing templates, first click Create a new template).
  3. Give the template a name. For example, "Monthly Newsletter".
  4. Below "HTML page", click Choose File to select the HTML file from your computer.
  5. Import the zip file (images.zip) Click Add template to upload the files.

Getting Started With Mailster

Here are few steps to get started with Campaign Monitor:

We assuming you have changed image background manually before uploading it to CampaignMonitor (see at the costumization section).

  1. Click on newsletter.
  2. Click on template.
  3. Click add new.
  4. Then you may have to attach the Mailster file of salut email theme.
